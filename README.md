# VoiceAgent

> AI Voice Agent Platform — AI-Powered Voice Receptionists for SMBs

## Overview

VoiceAgent provides small and medium businesses with AI-powered voice receptionists that answer calls, book appointments, handle FAQs, transfer calls, and take messages — 24/7, in natural conversational voice.

## Key Features

- **Natural Voice AI** — Human-like conversational voice powered by LLMs
- **Appointment Booking** — Real-time calendar integration and scheduling
- **Call Routing** — Intelligent transfer to the right department or person
- **FAQ Handling** — Answers common questions from your knowledge base
- **Message Taking** — Transcribes and delivers voicemails via email/SMS
- **Multi-Language** — Support for English, Spanish, Hindi, and more
- **Analytics Dashboard** — Call volume, resolution rates, peak hours

## Tech Stack

Python, FastAPI, Twilio, Claude API, WebSocket, React, PostgreSQL

## Getting Started

```bash
git clone https://github.com/MukundaKatta/voiceagent.git
cd voiceagent && pip install -e .
voiceagent setup --phone +1XXXXXXXXXX
```
